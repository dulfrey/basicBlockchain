## Basic Blockchain

Implementación básica en JAVA de Blockchain para el curso **Introducción a la Seguridad de la Información y Criptografía** en la **Universidad Nacional de Colombia**. Esta implementación solo muestra la estructura y comportamiento del blockchain.  
 
---

Basic JAVA implementation of Blockchain for the course Introduction to **Information Security and Cryptography** at the **Universidad Nacional de Colombia.** This implementation only shows the structure and behaviour of blockchain.
